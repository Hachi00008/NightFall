# NightFall

##  夜が、すべてを奪っていく。
##  生き残るほど、逃げ場はなくなる。

このゲームは、
昼と夜が交互に訪れるサバイバルPvPゲームです。

対応バージョン：1.21.11〜

昼は準備の時間。
PvPは無効化され、装備を整え、生き残るための対策をします。

夜になると、PvPが解禁され同時に安置が縮小し逃げ場が少しずつ奪われていきます。

日数が進むほど世界は過酷になり、
7日目以降は生存者全員に持続ダメージが与えられ、PvPが常に有効になります。

現在は、別のゲームモードは無く個人戦のみとなっております。

## ゲームの開始方法

まず、参加するプレイヤーと不参加のプレイヤーに分かれます。

参加・不参加の切り替えは以下のコマンドで行えます。

参加する場合：
/trigger Join_toggle set 1

不参加にする場合：
/trigger Join_toggle set 2

参加者が確定したら、以下のコマンドを実行します。

/function game:start

このコマンドを実行した位置が、
ワールドボーダーの中心となり、ゲームが開始されます。

ゲームを途中で終了・リセットしたい場合は、
以下のコマンドを使用してください。

/function game:reset

## ゲームランクについて

このゲームにはゲームランクというものが存在します。
順にTINY、SMALL、MEDIUM、LARGE、HUGEに加えてDEBUGの6モードあります。
ゲームランクは参加プレイヤーの数に応じて変更されます。


TINY

参加人数：2〜5

初期安置半径：150

夜ごとの縮小：-20


SMALL

参加人数：6〜9

初期安置半径：250

夜ごとの縮小：-30


MEDIUM

参加人数：10〜14

初期安置半径：350

夜ごとの縮小：-40


LARGE

参加人数：15〜19

初期安置半径：500

夜ごとの縮小：-50


HUGE

参加人数：20〜24

初期安置半径：650

夜ごとの縮小：-60


DEBUG

参加人数：0〜1

初期安置半径：150

夜ごとの縮小：-20

## 注意事項
※ ゲームは参加者が2人以上いないと開始・継続できません。


※ ゲーム中にログアウトしたプレイヤーは脱落扱いとなります。

※ 本ゲームは Minecraft Java Edition 専用です。

※ Bedrock Edition および Geyser 環境での動作は保証されていません。
